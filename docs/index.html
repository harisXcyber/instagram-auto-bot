How It Works

Step 1: Clone & Configure

git clone https://github.com/harisXcyber/instagram-auto-bot.git
cd instagram-auto-bot
pip install -r requirements.txt

Add your images to the images/ folder and configure your credentials in .env

Step 2: Deploy to Cloud

flyctl launch
flyctl secrets set INSTAGRAM_USERNAME=your_username
flyctl secrets set INSTAGRAM_PASSWORD=your_password
flyctl secrets set OPENAI_API_KEY=your_key
flyctl deploy

Deploy to Fly.io (free hosting) or any cloud platform of your choice

Step 3: Automate & Scale

Access your dashboard, set schedules, and let PrecisionFlow handle the rest. Monitor performance, adjust strategies, and watch your engagement grow.
