How It Works

Step 1: Setup (5 minutes)

Clone the repo and install dependencies:

git clone https://github.com/harisXcyber/instagram-auto-bot.git
cd instagram-auto-bot
pip install -r requirements.txt

Step 2: Add Your Content

Drop your images into the images/ folder. That's it.

Step 3: Configure

Create a .env file with your credentials:

INSTAGRAM_USERNAME=your_username
INSTAGRAM_PASSWORD=your_password
OPENAI_API_KEY=your_openai_key
DASHBOARD_PASSWORD=your_dashboard_password

Step 4: Deploy

Deploy to Fly.io (free hosting):

flyctl launch
flyctl secrets set INSTAGRAM_USERNAME=your_username
flyctl secrets set INSTAGRAM_PASSWORD=your_password
flyctl secrets set OPENAI_API_KEY=your_key
flyctl secrets set DASHBOARD_PASSWORD=your_password
flyctl deploy

Step 5: Relax

Your bot is now live. It'll post daily at your scheduled time. You can monitor everything from the web dashboard.

Tech Stack

Built with modern, reliable technologies:

  • Backend: Flask (Python web framework)
  • Scheduling: APScheduler (cron-like job scheduling)
  • Instagram API: instagrapi (unofficial but powerful)
  • AI: OpenAI GPT-4 Vision (image analysis + caption generation)
  • Database: SQLite (lightweight, no setup needed)
  • Deployment: Docker + Fly.io (free hosting)
